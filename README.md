# 📇 Contact Manager

A simple command-line contact management application built in Python.

## ✨ Features
- ➕ Add new contacts (name, phone number, email)
- 📋 View all saved contacts  
- 🔍 Search contacts by name or phone number
- 🗑️ Delete existing contacts
- 💾 Auto-saves to file (contacts persist between sessions)

## 🛠️ Technologies Used
- Python 3.x
- JSON for data persistence

## 🚀 How to Run
```bash
python contact_manager.py
